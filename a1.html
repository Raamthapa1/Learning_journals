    Describe your learning and experience creating sites using the two different CMSs.

  • Wordpress - I found Wordpress very easy to install. It has a easy interface. The major components such as pages, post and medias are easy to find and use. I found it easy to install plugins and themes.

  • Joomla - Joomla was also fairley easy to install, After installation It was bit tricky at fisrt to manage it. Installation of themes and plugins are also bit complicated because we have to do it manually.

    • Overall learning experience was really great. I explored powerful plugins, themes and features of both CMSs. I found Wordpress more user friendly and easy to use. But Joomla has more features and customization options. For example, when I was uploading a CEO message video to both my websites, I found that Joomla has more options to customize the video. I was able to change video frame size and also align it to centre. But in Wordpress I was not able to do that, the video was in original size and no option for alignment just for video. The inbuilt store for themes and plugin is great in werdpress, which i believe will be very handy for future projects. Wordpress is begineer friendly CMS whereas Joomla is more advanced and and will take time to learn.
